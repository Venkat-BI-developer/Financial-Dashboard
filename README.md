# Financial-Dashboard
**Project Name:**
Financial Dashboard for Revenue & Expenses

Business Purpose:
The Power BI dashboard is designed to provide an interactive overview of financial performance. It allows users to track revenue and expenses against budgets, analyze cash receipts, and gain insights into GL accounts to make informed financial decisions. The dashboard aims to simplify the financial reporting process and facilitate better financial forecasting and planning.

Tools Used:
Power BI Suites
SQL Server
Figma (to create custom canvas templates)

Datasources:
Excel
SAP BW
Description: Extracted data from SAP BO reports, SAP BW, SQL Server and sharepoint excel files and connected through PBI Gateway and deployed through Dataflow gen2 onto PBI Service.

Commectivity mode:
Import, Direct and Live Connections.

The dashboard was built using the following tools and technologies:
• 📊 Power BI Desktop – Main data visualization platform used for report creation.
• 📂 Power Query – Data transformation and cleaning layer for reshaping and preparing the data.
• 🧠 DAX (Data Analysis Expressions) – Used for calculated measures, dynamic visuals, and conditional logic.
• 📝 Data Modeling(Star Schema) – Relationships established among tables (Factreceivables, company, Date,customdate, customer, Salesrep, costcenter) to enable cross-filtering and aggregation.
